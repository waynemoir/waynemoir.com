.new Easing the path to a new world

What it is

The .new class is a generic container that serves as a means to limit the scope of the CSS styles in style.css.

Why is it needed

In an ideal world the latest and greatest styling would be simultaneously rolled out system wide, cascading perfectly and predictably across the application; this is a state that we are currently working towards.

The reality of the situation we are in is that there is not currently a consitant HTML structure used throughout the application. Controlling the scope of the new styling allows us to perform a graceful rollout across the system impacting only the areas we are ready to target.

How to use it

Locate the highest element which should be impacted by the new styling and wrap it in the .new container div. Hopefully this container will eventually be bumped up the tree; containing more and more of the page until it reaches the html element. Once every page in the system has reached this state this containing class will no longer be relevant and can be removed from the CSS and HTML entirely. Yay!

Markup

<div class="new">
    ...
</div>

Loading Feedback

Providing feedback whilst pages are loading makes the interface feel more responsive.

Ideally scoped feedback would be provided if the loading period is expected to be considerable. However in situations where we are unable to provide this, or for short loading periods, feedback should be given through the use of a looping animation.

Markup
