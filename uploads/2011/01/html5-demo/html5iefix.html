Testing HTML5 structure across browsers

January 14th 2011

The purpose of this page is to test HTML 5 elements across different browsers. This page contains all the structural elements that are new in HTML as of the 14th of January 2011. As the HTML specification has not been finalised there is a possibility that new elements could be added so this should not be seen as an exhaustive list.

The elements present on this page include:

					<article>
					<aside>
					<command>
					<figure>
					<footer>
					<header>
					<hgroup>
					<menu>
					<nav>
					<section>
					<time>
